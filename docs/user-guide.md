# 用户指南

## 功能特性

- **任务管理**：创建、编辑和管理待办事项
- **便签**：快速创建桌面便签，支持磁吸
- **定时器**：灵活的计时和提醒功能
- **截图**：快速截图并保存
- **截图翻译**：OCR 识别并翻译截图中的文字

## 系统要求

- Windows 10 或更高版本
- Python 3.8+（从源码运行）
- 建议 4GB 以上内存
- 300MB 以上可用磁盘空间（含 PaddleOCR 模型缓存）

## 安装

### 方式 1：运行已构建版本

进入 `dist/RedList/` 目录，双击 `RedList.exe`。

> 首次运行 OCR 功能时会自动下载 PaddleOCR 模型文件，需联网。
> 如启动失败，检查 exe 所在目录的 `报错日志.md` 了解错误详情。

### 方式 2：从源码运行

```bash
pip install -r requirements.txt
python main.py
```

## 配置翻译服务

应用支持多种翻译服务：
1. 百度大模型文本翻译
2. 百度通用文本翻译
3. 腾讯翻译君

### 配置步骤

1. 打开应用，进入"设置"选项卡
2. 在"翻译设置"区域选择翻译服务商
3. 输入对应的 API 密钥
4. 密钥会自动保存，下次启动时自动加载

### 获取 API 密钥

**百度翻译**：
1. 访问 https://fanyi-api.baidu.com
2. 注册并开通服务
3. 获取 APP ID 和密钥

**腾讯翻译**：
1. 访问 https://cloud.tencent.com/product/tmt
2. 开通机器翻译服务
3. 创建密钥获取 Secret ID 和 Secret Key

## 常见问题

**Q: API 密钥安全吗？**

所有密钥由用户自行输入，本地加密存储，不包含任何硬编码密钥，支持多服务商配置。所有数据本地存储，不上传用户数据，支持自定义数据路径。

**Q: OCR 识别不了？**

确保已联网（首次使用需下载 PaddleOCR 模型）。如果模型下载失败，检查网络代理设置。

**Q: 应用启动就崩溃？**

查看 exe 所在目录下的 `报错日志.md`，通常会包含完整的 Python 异常堆栈，可据此排查或提交 Issue。
