# Release Notes

## v3.0 (2026-04-03)

正式版稳定发布。

### 核心功能
- 任务管理：创建、编辑、删除待办事项
- 便签功能：桌面便签，支持磁吸和自定义
- 定时器：灵活的计时和提醒功能
- 截图工具：快速截图并保存
- 截图翻译：OCR 识别 + 多语言翻译

### 翻译服务
- 百度大模型文本翻译
- 百度通用文本翻译
- 腾讯翻译君

### 界面特性
- 简洁现代的用户界面
- 选项卡式布局
- 自定义设置
- 开机自启动选项

### 安全性
- API 密钥本地加密存储，无硬编码
- 所有数据本地存储，不上传用户数据
